База данных Northwind основана на вымышленной компании Northwind Traders. 

Categories - продуктовые категории и их описание
Contacts - контактные данные (поставщики, клиенты, сотрудники, специалисты служб доставки)
CustomerCustomerDemo - пустая таблица
CustomerDemographics  - пустая таблица
Customers - клиенты и вся информация по ним
Employees - иинформация по сотрудникам
EmployeeTerritories - соотнесение сотрудников и областей/территорий (к какой территории прикреплен сотрудник)
Orders - основная информация по заказам 
Order Details - дополнительная/детальная информация по заказам
Products - номенклатурный перечень продукции
Region - таблица регионов
Shippers - компании обеспечивающие доставку
Suppliers - поставщики
Territories - названия областей/территорий
